前言

​ 1、搭建和管理博客可能会需要大量的图片，而且这些图片还不能是本地的图片，因为图片资源太多会拖慢网页的加载，同样也不能是网上随便的图片链接，这样的不方便管理，所以我们需要一个自己的图床，供自己管理博客等网站的图片资源，然而国内没有比较优秀的免费图床，免费的那些图床存在各种限制， 收费相对于与一个小博客来说又是不必要的一笔输出。

​ 2、使用GitHub搭建+CDN搭建图床既满足了免费，又相对的保障了访问的速度，在此像发明这项技术的工作致谢。

搭建环境

  • CDN(jsDelivr)
  • PicGo](https://github.com/Molunerfinn/PicGo/releases) )
  • GitHub仓库

创建GitHub仓库

创建一个github仓库，专门存放上传的图片

为仓库起一个名字，点击Create repository完成创建

生成AccessToken

AccessToken用于免密码访问对应的GitHub用户，非常重要，千万不要泄露！

以上只需要按照图片顺序完成。

配置PicGo，使用jsDelivr的CDN

下载PicGo软件，安装。

打开PicGo进行配置

将刚才创建的仓库名填进第一项

将当前仓库使用的主分支名填进第二项

将刚才创建的Token填进第三项

将上传的路径填进第四项，设置的路径即使没有也会自动创建

自定义域名详细看下面。。

使用jsDelivr加速GitHub连接

格式：

https://cdn.jsdelivr.net/gh/GitHub用户名/具体的仓库名

例子：

https://cdn.jsdelivr.net/gh/Code-Neo/picBed

将设置好的链接设置到上面的最后一项，点击确定，就可以开始使用免费GitHub图床啦
